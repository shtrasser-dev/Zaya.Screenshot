Class CaptureService

Implementation of ICaptureService using Windows Graphics Capture API and Direct3D 11. Supports capturing windows and monitors in full-screen or rectangular regions. Call InitializeAsync(IReadOnlyDictionary<string, object?>?, CancellationToken) before creating sessions.
